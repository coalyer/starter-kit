# starter-kit

此项目作为记录学习和工作中遇到的`公共前端`和`专业前端`问题，并以 Demo 的形成呈现出来。<br>

> 本项目中的 demo 均为自己的学习`front end`之作，部分创意非原创。

## 安装依赖

```
npm install
```

## 运行服务

```
npm run server

open http://127.0.0.1:3030/menu.html
```

## 构建 menu 列表

```
npm run build
```
